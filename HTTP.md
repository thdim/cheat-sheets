# Common HTTP Response Status Codes:  
__200__ - OK  
__201__ - Created  

__400__ - Bad Request  
_bad supplied data from urls, headers or JSON_  


__401__ - Unauthorized  
__403__ - Forbidden  
__404__ - Not Found    
__405__ - Method Not Allowed  
__409__ - Conflict _(e.g. username exist)_    

__500__ - Internal Server Error
